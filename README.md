# VideoCut:通过字幕剪辑视频
VideoCut 通过基于OpenAI开源的大语言模型[Whisper](git@github.com:openai/whisper.git)为视频生成字幕,选择需保留的片段字幕重新生成剪辑后的视频
## 模型
![模型](imgs/image.png)
## 工作

- [ ] 通过ffmpeg对视频流进行读取
- [ ] 通过Whisper为视频生成字幕 (md格式或str格式)
- [ ] 通过字幕生成形成剪辑后视频
